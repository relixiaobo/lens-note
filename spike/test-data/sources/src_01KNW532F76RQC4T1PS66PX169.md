# 针对智能体的用户体验设计，第三部分：电子表格、生成式和协作式 UI/UX [译]

Reading notes and highlights from this source.

- 电子表格式界面（Spreadsheet UX）：一种基于表格布局的交互界面，其中每个单元格都可以作为独立的AI代理执行特定任务。
  - 保持了传统电子表格的熟悉操作方式，使用户能够轻松适应。
  - 特别适合处理批量工作，允许用户同时与多个代理交互，并支持数据扩充等任务。
- 协作式界面（Collaborative UX）：一种支持人类与AI代理实时协同工作的交互界面，类似于多人协作文档编辑器。
  - 其特点是支持并发操作，允许用户和AI同时工作并相互借鉴对方的成果。这种界面需要精细的变更管理机制，能够展示AI的实时工作进度，并能够合并并发修改。
- 环境式界面(Ambient UX)：一种让AI代理在后台持续工作的交互模式，其中用户可以专注于其他任务，而不需要直接参与AI的工作过程。
  - 最小干扰：不会打断用户的主要工作流程
  - 结果导向：主要关注工作成果的总结和变更展示
  - 触发机制：可以通过系统事件或webhook等方式自动启动
- 生成式界面（Generative UI）：一种动态生成用户界面的方式
  - 组件映射模式：将AI响应映射到预定义的UI组件上，确保界面的专业性和一致性
  - 完全生成模式：AI模型直接生成HTML代码来创建界面元素，提供最大的灵活性
