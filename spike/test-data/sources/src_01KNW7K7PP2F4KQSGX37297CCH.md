# 5000天后的世界

Reading notes and highlights from this source.

- 凯文·凯利认为社会变化的速度可能让我们在学校教育阶段学习到的技能变得无用，所以更加应该注重学习方法的掌握。
- 凯文·凯利认为，创新的本质是正确地对待失败，并引用硅谷的成果证明这一点
- 凯文·凯利认为，媒体过于关注负面新闻的报告，以至于扭曲了公众的正常认知。
- 凯文·凯利认为，对孩子而言，陪伴比物质更重要
- 凯文·凯利对未来的预测：镜像世界
- 凯文·凯利将"持续提问"的能力视为未来人类的核心竞争力，机器负责立即查询和产生结果。而教育应侧重培养提问的能力——这也正是通识教育的价值所在。
