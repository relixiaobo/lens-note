# MIT EI seminar, Hyung Won Chung from OpenAI. "Don't teach. Incentivize."

Reading notes and highlights from this source.

- 人类精心设计的"精巧结构"在大规模应用时往往会成为瓶颈。短期来看，精巧的结构可能会带来更好的结果，这种结构通常成为大规模应用的的瓶颈。长期来看，给模型更大的自由可能短期的效果并不理想，但如果给机器更大的自由去学习，会有非常好的潜力（长期有效的东西在短期内几乎必然看起来很糟糕）
- 精巧往往意味着复杂，稳定性差，不能适用
