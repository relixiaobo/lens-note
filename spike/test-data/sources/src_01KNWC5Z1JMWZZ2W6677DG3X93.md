# OpenAI 联合创始人 Andrej Karpathy 在2024年加州大学伯克利分校人工智能黑客马拉松颁奖典礼上的主题演讲

Reading notes and highlights from this source.

- Andrej Karpathy 建议进行以对外发布为目标的项目，有助于帮助大脑释放多巴胺，从而维持正反馈循环。
  - 项目发布后，他人的赞赏和认可，或为他人带来的帮助，都能极大增强成就感和满足感，进而产生愉悦的情绪。
  - 为了完成项目，通常需要针对性地学习和掌握所需的知识技能。相比广泛的学习，这种专注型学习，可以让所学知识得到立即实践和应用，从而带来更强烈的成就感。
  - 为了发布项目，通常会考虑不同受众的反馈，这促使我们不断提高工作标准，达到更优结果。适度难度的挑战和成功的体验都是激发成就感的重要因素。
