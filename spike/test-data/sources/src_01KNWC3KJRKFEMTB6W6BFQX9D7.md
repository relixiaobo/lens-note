# On Browsers, AI, and the web

Reading notes and highlights from this source.

- 作者认为 AI浏览器（Dia） 在商业上不可行
  - 用户本身不会为一个浏览器付费
    - 你的竞争对手有大量的预装
    - 你的竞争对手都是免费的
    - 你的竞争对手背后有大公司（Google、Apple、Microsoft）用其他业务利润来补贴
    - 最重要的是，你很难改变用户的使用习惯
  - 作为一家浏览器公司，Dia 全部基于 chromium 内核，只是其换皮应用
  - AI浏览器所依赖的AI服务主要依赖OpenAI或Claude的API，每次调用都有运营成本，并且没有技术门槛
- 作者认为AI聊天的交互并不会完全取代搜索结果页，成为用户与网络交互的默认中间层。
  - 很多网页中的创意和内容不能完全通过chat的形式呈现，只有聊天这种方式有点无聊；
  - AI的回答可能会因为背后的公司存在偏见；
  - AI只能提供一个只引用了前几条链接的主观答案，很可能有缺陷；
