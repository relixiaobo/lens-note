# 不要轻易地走进 AI Native（AI 原生）

Reading notes and highlights from this source.

- LLM 带来的是类似人类的智力（大脑），并且不需要像人类一样休息。
  - 类人的智力（输入 --> 处理 --> 输出）
    - 内容模态转换：在转换的过程中，补充信息或压缩信息，使其适合在对应模态下的消费。
      - 在 LLM 之前，我们压缩一个内容通常可行，比如将视频总结成文本，将论文总结成概要，但是反过来就很难做到，LLM带来这样的能力，在模态转换时，补充和扩展另一个模态所需的其他信息
      - 比如，通过添加语气、语速等信息，将书面文本转化为抑扬顿挫的声音，相比之下，如果仅仅是机械地将文本直接读出来，往往会产生单调乏味的效果。注意，其中添加的很多信息是AI无中生有的。
    - 知识和能力广博：通常大语言模型的预训练数据都包含了大量的人类知识，能做的事情很多，包括但不限于内容生成、翻译、编程辅助等
    - 创造性：得益于其结果的随机性，基本上其创造力是无限的，非常适合脑爆
    - 多语言能力：
    - 学习与适应能力：通过few-shot提示或fine-tuning的方式，学习原本不具有或不擅长的技能
    - 类推理能力：可以用于任务的理解与规划
      - 人类推理建立在经验、知识和逻辑思维的基础上
      - 大模型依靠海量数据训练得到的概率分布进行预测
  - 不需要休息
    - 批量生产：针对相同的查询或过程，也能批量生成具有随机性的结果
- 需求价值：即使支付体验很差，也不会减少地铁的乘客量。
- AI只是实现人类目标的手段，而不是目的，不要为了AI而AI，而是先知道人类要干嘛，再看AI能够在其中带来的帮助。
