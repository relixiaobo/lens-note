# 有一类Ai产品是帮人摘取书籍内容、缩短阅读时间的。 但凡产 - 即刻App

Reading notes and highlights from this source.

- AI总结可以用于满足"资讯阅读"的需求，但不能满足"理解阅读"的需求。
  - AI在辅助阅读防线有其局限性:
    - AI无法判断读者理解是否与作者产生歧义
    - AI被视为"全知全能"，不会误解作者意思
  - AI能做的不是帮你阅读，而是在你有主动阅读的意识下，辅助你更容易地阅读，比如帮你补充你与阅读材料之间的知识层次差，使你能够理解你原本不能够理解的内容。
