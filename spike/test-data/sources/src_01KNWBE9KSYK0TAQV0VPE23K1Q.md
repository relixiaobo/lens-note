# Man-Computer Symbiosis

Reading notes and highlights from this source.

- AI眼镜能够实时捕捉用户的视野，可以有效获取用户所处环境的上下文
  - AI系统会分析用户看到的内容，理解场景、物体、文字等信息。
  - 用户可以向AI眼镜提出指令或要求，可能是通过语音或其他输入方式。
  - AI眼镜会继续观察这些反馈，形成一个闭环系统。它可以验证操作是否成功，或者根据新的情况调整后续动作。
  - 基于用户的视野内容和指令，AI眼镜能够帮助用户操作其他电子设备。这可能包括智能手机、电脑、智能家居设备等。其他设备执行操作后会产生反馈，比如屏幕上显示的内容变化、设备状态改变等。
