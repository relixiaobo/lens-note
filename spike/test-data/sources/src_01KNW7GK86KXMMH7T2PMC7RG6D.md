# 2万字长文，如何成为一个“懂”AI 的产品经理？

Reading notes and highlights from this source.

- 大模型的性能、成本和延迟构成了一个互斥的不可能三角，提升模型性能通常需要更大的参数量，这既会增加运算成本，也会导致更长的响应时间。这三个属性之间存在此消彼长的关系，无法同时达到最优。
- AI在工作场景先得到应用，是因为AI的成本问题。工作场景一般产生的价值足够大，人们更愿意为其付出金钱和尝试的成本。
- 大模型的上下文窗口是有限的，并且超出"有效工作窗口"的长度后，回答的准确率会直线下降，目前产品上的一个挑战就是让两者达到平衡的可接受状态。
- 基于大模型进行产品设计，更需要考虑的是为用户带来愿意付费的价值，而不是弥补模型的局限。
