# 提问：AI 对您写笔记和进行知识管理有怎样的帮助？

Reading notes and highlights from this source.

- 使用AI来辅助自己执行费曼学习法，让其AI变成一个听众，然后你去给他讲你学到的内容或思考出来的观点，然后让AI扮演一个完全不懂的小白来给出反馈，或者提出AI可能觉得不能理解的地方，这样来不断地优化表达，发现自己理解中的盲点或不足。
