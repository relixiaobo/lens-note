# The End of Organizing

Reading notes and highlights from this source.

- 组织笔记的过程带来了了理解笔记的必要难度，也许会带了更多的思考。对于某些人群来说，也许思考才是最重要的记笔记的原因。
- 记笔记以用途为目的的用户，也许AI可以帮上忙。因为AI可以做更好的提示，在正确的时间、以正确的格式，帮你找到并有效使用你的旧笔记。
- 为什么组织笔记和整理笔记如此困难？
  - 记录笔记的目的或者用途通常很多，很难找到一个单一的系统满足所有的用途。
    - 这可能导致了我们不断尝试新的系统，因为当前的系统可能不能满足某个突然出现的需求。
  - 在需要用到的时候，旧笔记往往都没准备好（看起来像是老旧的垃圾），因为需要理解记录旧笔记的上下文，并且要找到跟新的事情的联系，甚至需要重写，所以使用旧笔记的代价太高，不如直接找新的。
    - 这可能导致了我们很可能不会去看旧笔记，因为代价比收益要高。
  - 通常，我们记下一条笔记的时候，并不知道这条笔记可能的用途。
