# 复杂工程化最终会被模型碾碎

Reading notes and highlights from this source.

- 很多我们现在正在做的事情，可能模型进化之后就一下子变得没有价值了，所以我们更是必须看清楚什么是短期要做的，什么是我们的长期目标
  - GPT-4o在图像生成领域的应用，让原本很多为图像生成进行优化的产品一下子变得可有可无
  - 精心调教的提示词、层层叠加的插件、环环相扣的模型链——如今都被一个简单对话界面所取代
