# 古典产品拼命通过各种手段来激发多样性，AI 产品则拼命通过各种手段来控制多样性。

Reading notes and highlights from this source.

- AI产品与传统产品对于多样性的处理策略差异
  - 技术特定对比
    - 传统产品基于确定性的代码逻辑，所以天然就会有稳定的特性
    - AI产品基于概率模型和大规模数据训练，结果随机是必然的结果
  - 核心观点：AI产品和传统产品在处理多样性方面采取了不同的策略，这主要由它们背后的技术特性所决定。
  - 为了提供稳定与惊喜的平衡体验，所以传统产品的发展方向是在稳定的基础上激发多样性，AI产品的发展方向是在随机的基础上寻找稳定性。
