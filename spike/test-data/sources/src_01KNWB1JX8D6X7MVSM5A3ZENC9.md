# julian.: 张小龙在一次直播连麦里回答了很多关于那次微信更新的问题，本身问题都是相对无聊的八卦类问题，但他很不经意的说了句认为视频尤其是长视频的消费结合朋友推荐或许是可以探索的，因为长视频的消费成本高。当时这句话没有被主持人重视和追问，但我记这句话还蛮深的。 确实今天的电影其实很多部分是这样的（看朋友去看了因此也去看，并看完在社媒上分享），b 站上的 30 分钟以上的视频为什么不能这样？以一种更社交推荐内容的机制。

Reading notes and highlights from this source.

- 消费成本低的，使用推荐算法推流，内容或者服务会被直接消费；如果是消费成本高的内容或商品被推荐算法推流，用户往往需要决策，是在以后再进行消费还是现在马上消费，但如果是朋友（社交网络）推荐，往往会让消费者更加信任。
