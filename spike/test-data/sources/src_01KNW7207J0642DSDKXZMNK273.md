# 我看见的世界

Reading notes and highlights from this source.

- 并不是每一次技术升级带来的都是价值，有时候带来的可能是另一个更大的问题。技术设计若只看功能不看人，易把问题放大。比如，之前医疗行业的IT升级，就带来了很多信息过载的问题，导致医务人员不能识别真正有价值的信息，并且很容易疲累。
- 李飞飞认为数据是现代视觉系统的养分，人类是借助"经验"来理解见到的一切全新事物的，ImageNet 之所以有效，也是因为学习了成千上万的人类"经验"（带标签的图片）。
- 李飞飞认为，人工智能应该致力于提高人类的能力，帮助人类，而不是与人类竞争。如何实现这一点，应该从人类自身的角度开始思考，而不是从技术的角度开始思考。
- Ambient Intelligence (环境智能): 将智能融入背景之中，并在需要的时候给出提示。
- Edge Computing (边缘计算): 现在可用的人工智能都是云端的，但总有很多场景是隐私和安全优先的
