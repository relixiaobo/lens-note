# 6000字+6个案例：写给普通人的MCP入门指南

Reading notes and highlights from this source.

- MCP协议统一了应用或服务向LLM传输数据的方式，降低了跨模型工具的开发的困难程度。
- MCP让普通人能够定义自己的工具以及工作流的门槛降低（试图解决软件之间的信息孤岛问题）
  - "被服务"指的是只能在用户提供明确指令后才能执行任务，被限制在特定平台或界面中，无法主动连接其他服务或工具
  - "主动服务"指的是成为调用其他工具和服务的"协调者"，可以自主调用各种外部工具和服务，允许普通用户像搭积木一样组装自己的AI工作流
