# 针对智能体的用户体验设计，第二部分：后台运行[译]

Reading notes and highlights from this source.

- 如何建立起人类对 Agent 的信任？
  - 主动寻求人类帮助
  - 过程透明：允许用户实时查看 Agent 的决策和执行过程，如果需要也可以回看整个过程
  - 允许纠错：允许用户回到特定步骤进行修正，包括直接修改错误操作、指导智能体改正，或是调整初始条件重新运行
- 后台运行的 Agent 相比于流式聊天的大模型的优劣势分析
  - 并行处理优势：后台运行模式允许多个Agent同时处理多个任务，突破了聊天界面一次只能处理一个任务的限制。
  - 任务质量提升：当AI在后台运行时，用户对完成时间的要求会更宽松，这让AI能够更专注于提供高质量的结果，而不是被迫快速响应。
