# What makes a great ChatGPT app

Reading notes and highlights from this source.

- 优秀 ChatGPT App 标准：是否在基础模型能力之外，提供了以下三种额外的能力？
  - 做： 它是否允许 ChatGPT 代表用户采取实际行动？
    - 在这里，应用程序与其说是真相的来源，不如说是一双手。它接收模型的意图，并将其转化为你的团队日常使用的系统中的具体变更——或者在游戏的情况下，转化为游戏状态中的具体变更，从而使体验感觉连贯且公平。
  - 展示效果：相比纯文本，是否以更清晰、更便于操作的界面呈现信息？
    - 应用程序可以为模型提供一种结构化语言：包含列、行、分数的小部件，以及与人们实际决策方式相匹配的视觉元素——或者在游戏中，帮助他们理解"自己在世界中的位置"。#highlight
  - 了解：它是否为 ChatGPT 提供了原本无法获取的新背景信息或数据？
    - 应用程序成为模型在你所在领域的"眼睛和耳朵"，使其能够比通用文本训练更权威地回答问题。
- 在真实的 ChatGPT 对话中，你的应用很少是唯一在运行的。模型可能会在同一次对话中调用多个应用。从用户的角度来看，这是一个流程。从你的角度来看，这是在提醒你，你是生态系统的一部分，而不是一个封闭的产品。让其他应用或者ChatGPT能够轻松地利用你的输出，使得用户受益。
