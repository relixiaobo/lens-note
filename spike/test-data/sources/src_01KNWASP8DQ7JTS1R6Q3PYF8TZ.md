# Introducing Semantic Reader

Reading notes and highlights from this source.

- 阅读论文时，阻碍理解流畅性的摩擦点
  - 无法追踪阅读历史和笔记
  - 应对PDF格式，这种格式不适合移动阅读或辅助技术（如屏幕阅读器）
  - 经常来回翻页寻找引用论文的详细信息
  - 难以在多篇论文中识别同一项研究
- Semantic Reader 为了让用户更快速地浏览论文，使用AI生成的高亮找出论文的关键点，包含三个类别标签：目标(Goal)、方法(Method)和结果(Result)。
