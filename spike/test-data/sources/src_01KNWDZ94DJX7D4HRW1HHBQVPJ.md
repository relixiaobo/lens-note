# What's next for AI agentic workflows ft. Andrew Ng of AI Fund

Reading notes and highlights from this source.

- Agent 工作流程的四种主要设计模式
  - 反思（Reflection）：通过让语言模型（LM）自我检查和反馈来改进生成的代码或内容。例如，生成代码后，提示LM检查代码的正确性、效率和结构，并根据反馈进行改进。
  - 规划（Planning）：使用规划算法让AI代理能够自主决定执行任务的步骤，并在遇到失败时重新规划路径。这种方法可以显著提高AI代理的自主性和灵活性。
  - 利用工具（Tool Use）：使用语言模型结合各种工具来扩展其功能，例如进行网页搜索、生成和运行代码等。这种方法最早在计算机视觉领域得到应用，因为早期的语言模型无法处理图像。
  - 多智能体协作（Multi-Agent Collaboration）：通过多个语言模型代理之间的协作来完成复杂任务。例如，一个代理负责编码，另一个代理负责代码审查，或者通过不同代理之间的辩论来提高性能。
