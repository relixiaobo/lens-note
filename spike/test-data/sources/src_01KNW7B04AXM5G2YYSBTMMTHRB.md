# 美丽新世界

Reading notes and highlights from this source.

- 人既需要社交，又需要独处。无论是极度缺乏社交机会，还是过度强制参与社交，都会导致痛苦。
  - 在熔岩区（原始社会）：
    - 排斥感：野人被排除在集体活动之外，这引发了强烈的孤立感和被拒绝感。
    - 归属需求：人类本质上是社会性动物，有强烈的归属需求。被排斥意味着这种基本需求得不到满足。
    - 身份认同：参与集体活动往往是建立身份认同的重要方式。无法参与可能导致身份危机。
    - 社会地位：在原始社会，参与集体活动可能与社会地位紧密相连。被排斥可能意味着低下的社会地位。
  - 在文明的伦敦：
    - 文化冲突：作为"野人"，他可能难以理解和适应这种高度组织化的社会形式。
    - 思考时间不足：缺乏独处时间意味着没有足够的时间进行深度思考和自我反省。
    - 隐私缺失：无法获得独处时间，意味着个人隐私和自我空间的严重缺失。
    - 个性压抑：过度的集体活动可能导致个性被压抑，无法真实地表达自我。
    - 自主权丧失：强制参与集体活动剥夺了个人的选择权和自主权。
  - 任何试图将人性简单化或统一化的社会制度，会给人带来痛苦。
- 教育（小说中是"培育"），可以让人们追求"普世幸福"，即认为幸福和舒适比真理和美更有价值，有助于维持社会的稳定。比如，消费主义，买更多更贵的物品就是幸福。
- 对于控制和影响行为，奖励比惩罚更有效。
  - 正反馈（奖励），带来行为习惯的养成和强化。
  - 惩罚暂时能够停止不合乎要求的行为，但无法永久消除被惩罚者沉溺其中的念头。
- 随着年龄渐长，很多开始相信宗教，因为相对于激情、幻想、感官刺激、欲望等，他们开始有了更高层次的精神追求，如生命的意义、自我的本质、道德伦理等。而宗教为这些问题提供了一个现成的、系统化的答案和追求路径。
- 对于世界的不满意，可能会因为个体的成功而改变，但其实世界并没有任何改变。批评会让人自我感觉良好。
  - 与他人一起批评，可能会获得群体认同
  - 批评他人或事物时，人们往往会感到自己站在了道德或智力的高地上。这种优越感会增强自尊，让人感觉良好。
  - 通过批评，人们可能感觉自己在掌控局面或有能力改变事物。这种控制感能带来安全感和满足感。
  - 批评他人或事物可以是间接肯定自己价值观的一种方式。
  - 批评可能吸引他人的注意，如果得到认同，会带来被认可的满足感。
  - 批评可能是行使新获得权力的一种方式。
- 幸福是现实结果与个人期望之间的差值，但是个人期望是有可能被社会的引导所塑造的
