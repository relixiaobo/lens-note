# 科技爱好者周刊（第 363 期）：最好懂的神经网络解释

Reading notes and highlights from this source.

- 认知债务 (cognitive debt)：人工智能在带来便利的同时，但是长期代价是思考、认知能力的变弱。
  - 当下很爽：几乎不用思考，就能得到结果，效率爆表
  - 以后要"还"，还要付出"利息"：很难记得之前跟LLM一起做过什么？解决方案是什么？后续在没有AI的情况下，很难静下心来，独立思考，解决问题
  - 解决办法：先自己思考，再让大模型验证自己是否出错，而不是先用大模型去解决"困难"的部分。
