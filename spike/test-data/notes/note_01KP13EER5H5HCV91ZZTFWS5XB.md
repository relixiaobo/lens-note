A CLI tool designed for agent consumption should have at most 5 core commands. Research: fewer than 20 tools per turn (OpenAI), coarse-grained beats fine-grained (Berkeley ToolPlanner), tool definitions consume 5-7% of context window (Klavis). One unified write command with JSON type routing is more agent-friendly than separate create/update/link/delete commands.
