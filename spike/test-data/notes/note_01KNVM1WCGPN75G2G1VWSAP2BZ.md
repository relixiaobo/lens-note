The spreadsheet as multi-agent UX is a fascinating convergence: the grid is a pre-existing mental model that maps cleanly onto parallelized agent execution. Each cell = one agent task = one token budget. Users already understand cells as independent, composable, and scalable. This is not just a UI choice — it's a familiar *abstraction* that lets non-technical users reason about concurrency without needing to understand it. The spreadsheet interface may become the dominant paradigm for batch AI interaction for the same reason it became dominant for data: it makes parallel structure legible.
