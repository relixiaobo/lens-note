There is a structural reason why the chat format cannot fully replace the search results page: content diversity. The web contains creative works, interactive tools, visual media, and navigable information structures that are not reducible to a synthesized text answer. Chat flattens all of this into a single response modality. This is not a limitation of current AI — it is a limitation of the format itself. The implication: AI chat is well-suited to factual lookup and synthesis tasks, but the search results page will persist as the appropriate interface for discovery, exploration, and content that is *inherently* non-textual. The claim that AI chat becomes the 'default middle layer' between users and the web confuses 'AI improves search' with 'chat replaces search.'
