The Anthropic Barry/Mahesh analogy makes a sharp point that cuts against the intuition of 'smarter = better agent': for domain-specific tasks, a 300-IQ generalist who must reconstruct domain knowledge from first principles is consistently outperformed by an experienced specialist. This is an argument about where agent value comes from — not raw reasoning capacity, but accumulated, pre-loaded domain expertise. The implication for agent design: intelligence is table stakes; the differentiator is *situated knowledge*. This maps onto a real AI failure mode: highly capable models that perform poorly on specialized tasks because they're reconstituting domain knowledge in-context every time rather than having it pre-crystallized.
