Relaxed time expectations in background mode create a quality dividend: when users don't expect an immediate response, the agent can invest more compute and reasoning depth. This is the inverse of the streaming-chat pressure toward fast, shallow responses. The implication is that the same underlying model may produce qualitatively different outputs depending on the UX contract — not because its capabilities change, but because the socially-negotiated time budget changes. This suggests that task quality is partially a function of interaction mode, not just model capability.
