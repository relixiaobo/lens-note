Perplexity's Comet Browser is also a direct challenge to the longitudinal behavioral context moat described in note_01KNW9VZ9K7JV80QKFRQ00HP6Z. That note argues that 'access to longitudinal behavioral context is structurally unavailable to any agent that lives outside the platform.' Perplexity is betting that this structural disadvantage can be eliminated by building the platform itself — i.e., owning the browser to own the history. If successful, Comet would be the first case where a third-party AI company closed the platform moat gap not by integration, but by vertical expansion into the platform layer. The strategic question is whether browser market share is achievable without Google/Apple's OS control, which creates a secondary distribution moat Perplexity still doesn't own.
