SceneScout — AI-driven access to street-level imagery for blind users — represents a category of AI application the existing knowledge base hasn't addressed: AI as a sensory bridge for users whose perceptual channels are structurally blocked from certain information. This is categorically different from AI as a productivity tool or AI as an agent completing tasks. Here, the AI's role is epistemic mediation — translating visual information that the user literally cannot access into a form they can. The interesting design question this raises: who defines what's 'relevant' in a visual scene when the user cannot verify the selection? The AI's editorial choices about what to describe become invisible filters on the user's world model.
