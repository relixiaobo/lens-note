There is an open question about whether the Bitter Lesson applies to the current LLM era or whether we are already in the 'computation wins' phase. If LLMs represent the triumph of scale-over-knowledge, then the new Bitter Lesson might be about something else — perhaps about architecture choices, or RLHF-style alignment, or agent scaffolding — wherever the next human-knowledge bottleneck gets inserted.
