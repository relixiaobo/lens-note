Chat's natural error-correction affordance may be its most underappreciated advantage over other Agent UX modes. When a large model misunderstands, the user can simply say 'no, I meant X' — the same repair mechanism humans use in conversation. Spreadsheet, Ambient, or Generative UI have no equivalent lightweight correction loop: fixing a misunderstanding requires re-prompting, re-running, or re-designing. This asymmetry suggests Chat UX is not just simpler — it is more *fault-tolerant by design*, which matters especially in early-stage agent deployments where model errors are frequent.
