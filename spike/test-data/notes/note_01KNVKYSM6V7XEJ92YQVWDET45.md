Agent anatomy (LLM + retrieval + tools + memory) reveals that the base model is just one component — the scaffolding around it is what determines what the agent can do. This means capability improvements can come from two independent directions: improving the base model, or improving the scaffold. The Bitter Lesson predicts that base model improvement will eventually dominate. But the scaffold may be doing something irreducibly different — routing to external state (memory/retrieval) and external action (tools) — that cannot be absorbed into the model weights alone.
