There is a structural parallel between the two Generative UI modes and the Bitter Lesson's core tension. Component mapping mode = injecting human design knowledge (predefined components = hand-crafted structure). Full generation mode = letting the model discover its own representation. The component mapping mode may feel safer and more consistent — but by the Bitter Lesson's logic, betting on AI's ability to generate appropriate interfaces from scratch may ultimately win. The question is whether visual/UX design has the same 'more computation always wins' property as cognitive tasks, or whether there are genuine human-preference constraints that resist learning.
