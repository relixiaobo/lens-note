Agent workflow patterns are themselves a form of human-knowledge injection into LLM systems — exactly the kind of scaffolding the Bitter Lesson warns about. Prompt chaining, routing, orchestrator-worker: these are human-designed control structures encoding assumptions about how tasks decompose. The open question is whether better base models will eventually render these patterns obsolete (as they did hand-crafted NLP pipelines), or whether workflow structure is a different kind of thing — coordination infrastructure rather than cognitive knowledge.
