「所有 LLM 行为都是幻觉，有用的部分只是我们发现的子集」——这个论断把 LLM 的「创造力来自随机性」（note_01KNW6PDXC2ZPAKBG1KQ3TSHC1）和「信息扩展」（note_01KNW6PDX42NY6ZMT6GJ7BFTF1）两个命题推向了它们的逻辑极端。那两个笔记已经在说：LLM 的输出是从分布采样而非从真理推导。但它们仍然暗含了一个区分：有些输出「恰好」是对的，有些是创造性的，有些是错的。而这篇文章的观点直接消解了这个区分：不存在「恰好正确」，只存在「人类认为有用」。这是一个认识论上的激进立场：LLM 没有「知道」任何事情，它只有生成——我们在这些生成物中提取价值，就像从噪声中挖掘信号。
