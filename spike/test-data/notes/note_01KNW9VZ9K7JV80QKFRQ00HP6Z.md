Gemini in Chrome accessing browser history to find previously visited pages introduces a qualitatively new kind of agent memory: longitudinal behavioral context. Existing notes discuss working memory (context window, skill loading) and task-level memory (rollback/replay). But browser history is neither — it's a passive, continuously-accumulated record of the user's attention over weeks and months. An agent that can query this history is not just better at tasks; it has a user model that no standalone AI assistant can build without platform integration. This is the core competitive moat of OS/browser-native AI over third-party agents: access to longitudinal behavioral context is structurally unavailable to any agent that lives outside the platform.
