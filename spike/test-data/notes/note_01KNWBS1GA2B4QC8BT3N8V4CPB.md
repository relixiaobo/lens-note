The human cognitive taxonomy of memory — episodic, semantic, procedural — maps cleanly onto agent architecture, and the mapping reveals an asymmetry: agents today can update episodic memory (interaction history) and semantic memory (knowledge facts), but procedural memory (LLM weights + agent code) is frozen at deployment. This is not merely a technical limitation — it's a fundamental distinction between agents that *remember what happened* and agents that *become different*. The implication is that current agents can personalize within sessions, but cannot genuinely learn to be better agents. All adaptation today is retrieval-based, not weight-based.
