协作式 UX 与环境式 UX 的核心区别在于并发性：协作式 UX 中人与 LLM 同时工作、互相借鉴；环境式 UX 中 LLM 在后台持续运行，用户专注于其他事情。两种模式对应不同的技术需求（细粒度输出展示 vs. 事件触发与变更摘要）。
