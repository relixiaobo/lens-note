Sutton's second point — that the actual contents of minds are 'tremendously, irredeemably complex' — has an important implication for AI interpretability/explainability that the existing notes don't capture. If the world (space, objects, agents, symmetries) is not just complex but *intrinsically* complex — complexity without a simpler underlying structure waiting to be found — then interpretability efforts that seek 'simple ways to think about' model internals are committing the same error as knowledge injection. Explainability, as usually conceived, assumes that a comprehensible representation of what a model 'knows' exists and is recoverable. The Bitter Lesson's second point suggests this assumption is false: there is no simple representation, because the territory itself has none.
