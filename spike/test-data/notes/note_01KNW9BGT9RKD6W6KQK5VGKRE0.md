The 'day 30 better than day 1' framing names the cold-start problem as the central unsolved problem of deployed AI agents. Every time a capable model starts fresh, it arrives without accumulated knowledge of that user's domain, workflows, and edge cases — it is, in the Anthropic speakers' analogy, perpetually the first-day employee. Skills are proposed as the solution: crystallized expertise that persists across sessions, so the agent carries forward what was learned without requiring re-explanation. This is conceptually the same thing memory does, but with a crucial difference: memory stores *what happened*, while skills store *how to do things*. Skills are episodic memory compressed into procedural memory — the difference between 'I remember we did X' and 'I know how to do X.'
