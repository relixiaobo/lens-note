The 'high variable cost' criterion for direct AI monetization reveals the structural reason most AI features cannot be directly monetized: inference costs scale with usage, but only features where users consciously perceive value per use can bear per-use pricing. For AI features that improve background metrics (retention, engagement), there is no usage event salient enough to anchor a charge to. This means the variable cost problem and the perceived-value problem are not two independent criteria — they are two sides of the same coin. A feature the user doesn't consciously use cannot be charged per-use; a feature whose cost you can't cover indirectly must be charged per-use. The cases where direct monetization fails are exactly the cases where usage is invisible to the user.
