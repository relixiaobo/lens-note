「异步人机协作」补全了 session handover 论断缺失的另一个维度。现有知识库把长时运行 agent 的问题定义为「项目交接问题」（agent 内部的时序连续性）。这篇文章提出的任务/Issue 系统（Linear 形态）补全了另一层：不只是 agent 如何记住自己做了什么，而是人类和 agent 如何在异步时间流中共享任务状态、互相创建和流转工作。两者合起来构成完整图景：session handover 解决 agent 的时序连续性，任务系统解决人机之间的协作协调。
