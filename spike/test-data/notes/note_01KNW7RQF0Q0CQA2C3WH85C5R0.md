「信息孤岛」是 MCP 想解决的问题，但这个问题的根源其实不是技术而是经济激励。每个平台制造信息孤岛，是因为封闭数据对平台有利（用户锁定）。MCP 降低了技术互通的成本，但没有改变平台封闭数据的动机。真正的问题是：当互通对用户有利、但对平台有害时，协议标准化能走多远？历史上，开放标准（如 email/SMTP）在平台没有太强封闭动机时成功；但在平台具有强锁定动机的领域（如社交图谱），开放标准屡屡失败。MCP 能不能真正打破信息孤岛，取决于工具生态的商业模式，而不只是技术标准的优雅程度。
