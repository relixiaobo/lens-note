The 'Powerful' pillar (code, image generation, video via Veo, web interaction via Project Mariner) is essentially the AI-as-universal-tool claim: a single assistant that replaces a portfolio of specialized tools. Historically, specialization wins quality competitions. Google is betting that the coordination cost of managing multiple specialized tools exceeds the quality gap between a universal assistant and specialized tools. This bet is only reasonable if: (1) the models are good enough across domains, and (2) the integration via Personal Context makes the universal assistant's outputs *superior* (not just more convenient) to specialized alternatives. Personal context is the proposed lever for turning 'good enough at everything' into 'better for this specific user than any specialist.'
