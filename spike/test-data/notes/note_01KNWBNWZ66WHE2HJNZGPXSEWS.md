Arc II illuminates a category distinction that the Ambient UX frame collapses: there are tasks where the *process* is the value (exploratory browsing, serendipitous discovery), and tasks where the *outcome* is the value (finding a specific fact, comparing prices). Arc II aggressively optimizes for the latter while eliminating the former. For users who browse instrumentally, this is a pure win. For users who browse experientially, this is a category destruction. This is a direct instantiation of the warning in note_01KNW7EG7XAM0BZFYC6BF3NRYF: optimizing for a single dimension (task completion efficiency) annihilates the value that lives in friction, surprise, and wandering presence.
