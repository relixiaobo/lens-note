The paper's out-of-vocabulary (OOV) word recovery result is conceptually important: embedding distances achieve accuracy within 0.5 percentage points of phone edit distances. Phone edit distance is a linguistically-informed, human-designed similarity metric. The embedding distance is learned. The near-parity suggests the embedding has essentially rediscovered the phonological distance metric that linguists designed from first principles — without being explicitly told to. This is the same pattern as LaCy, where a learned system rediscovers something that was already known via explicit rule systems. The question this raises is whether the embedding has 'discovered phonological distance' or merely found a functional proxy that happens to correlate. These are different claims with different implications for generalization.
