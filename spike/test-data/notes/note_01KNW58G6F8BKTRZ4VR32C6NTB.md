Non-streaming chat and Ambient UX share a structural similarity that disguises a crucial difference. Both deliver results without showing intermediate work. But Ambient UX earns this by being 'opt-in background' — the user explicitly dispatches a task and returns later. Non-streaming chat *looks* like synchronous interaction but withholds process visibility, creating a frustrating middle ground: the user is waiting, engaged, and getting nothing. The failure mode of non-streaming chat is that it inherits Ambient UX's opacity without its asynchronous social contract.
