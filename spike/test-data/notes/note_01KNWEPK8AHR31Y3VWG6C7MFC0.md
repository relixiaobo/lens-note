如果 workflow 不确定但 context 确定（输入清晰但走法多样），Agent 需要自主规划路径。例如市场分析报告生成、个性化推荐等任务。大多数 End-to-End RL Agent 擅长此类任务，因为训练阶段习得了大量路径规划和解题思路。
