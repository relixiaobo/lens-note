LLM 的「创造力」是随机性的副产品，而非独立的能力。文章将 LLM 的创造性归因于「结果的随机性」——这是一个值得认真对待的观察：LLM 的创造性并非来自某种神秘的「理解」，而是因为相同输入每次都从概率分布中采样不同结果。这与既有笔记 note_01KNW6F6ENER70M6JZ6P94NNM5 的论点形成呼应：AI 产品用各种手段「控制多样性」，正是在压制这种采样随机性。换言之，AI 产品中「可靠性」与「创造性」是同一枚硬币的两面——压低温度提升可靠性，同时扼杀创造力。
