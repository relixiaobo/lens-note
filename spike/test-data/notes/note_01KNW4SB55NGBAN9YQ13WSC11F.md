如果 workflow 确定但 context 不确定（流程固定但输入多变），就需要 Agent 在语义和理解上补全——例如客服问答、合同解析——通过外部检索、知识图谱等工具来弥补信息缺口，让推理结果更符合预期。(When the workflow is fixed but the context is variable, agents must fill semantic gaps — as in customer service Q&A or contract parsing — using external retrieval, knowledge graphs, and similar tools to make reasoning outputs match expectations.)
