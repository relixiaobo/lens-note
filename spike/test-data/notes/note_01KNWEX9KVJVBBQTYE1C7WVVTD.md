当 workflow 和 context 都不确定时，是最复杂的 Agent 场景，既要推理也要探索（如创新方案设计、跨部门信息收集）。这类通用型 Agent 的执行效果取决于配备的工具丰富度，尤其需要最大化开放编程能力——例如让它学会去 GitHub 找仓库、克隆并修改代码来解决问题，让它像人一样干活儿。
