Arc II's 'browser that browses for you' is the Proactive AI pattern (note_01KNW9ZSTAQJ6JG9RVFBG1AB4W) applied specifically to the browser's core function — navigating and reading web pages. But what this framing reveals is a deeper architectural claim: when AI takes over the *search → click → read → synthesize* loop, the browser ceases to be a navigation tool and becomes a synthesis delivery mechanism. The 'browsing' step, which was previously the user's cognitive work, is delegated wholesale. This is structurally more radical than booking an appointment via Chrome — it replaces not just one task, but the defining behavior the browser was built around.
