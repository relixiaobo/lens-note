The leaked OpenAI playground meta-prompt is self-exemplifying: it is itself structured according to the template it prescribes (Concise instruction first → Details → Steps → Output Format → Examples → Notes). This is not coincidence — a prompt that generates prompts must be its own best instance, or it will teach by contradiction. This self-referential coherence is a reliability signal: if the meta-prompt violated its own structure, the model being instructed would receive contradictory evidence about what good structure looks like. The principle generalizes: any instructional artifact that is also an instance of what it instructs gains credibility from enacting its own advice, and loses credibility if it fails to.
