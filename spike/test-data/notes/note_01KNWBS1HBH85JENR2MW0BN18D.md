The episodic memory framing for agents — 'past interaction sequences including user preferences' — is a cleaner cognitive label for what note_01KNW9VZ9K7JV80QKFRQ00HP6Z calls 'longitudinal behavioral context.' The cognitive science framing adds something: it clarifies *why* episodic memory matters for agents — not just because it stores facts, but because episodes encode contextual, preference, and value signals that factual knowledge bases don't capture. A user's repeated choices across interactions reveal their values. This is what human episodic memory does too — it is not mere record-keeping, but the substrate from which personal identity and preference are constructed.
