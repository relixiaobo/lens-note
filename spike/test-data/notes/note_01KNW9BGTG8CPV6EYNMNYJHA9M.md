The "repeatedly training the same brilliant but inexperienced new hire" framing reveals a hidden cost of stateless agents that is usually invisible: the cost is borne by the user, not the system. When an AI agent re-derives domain context every session, the user pays the onboarding tax — they must re-explain, re-specify, re-contextualize. Skills transfer this cost from the user to the system: expertise is captured once and reused infinitely. This is structurally identical to the argument for good documentation and onboarding processes in human organizations — the cost of writing good docs is a one-time investment; the alternative (verbal re-explanation) is a recurring tax paid at every handoff. Skills are documentation for agents.
