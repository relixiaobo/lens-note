Gemini in Chrome operates across multiple open tabs simultaneously — which is structurally the same as what note_01KNW5AYYTAAGD1CRF28CKE353 calls the parallel execution advantage of background agents. But there is a subtle difference in framing: that note describes users becoming 'dispatchers managing a portfolio of concurrent executions' (many agents, many tasks). Chrome's multi-tab Gemini is the inverse: one agent, aware of many tabs. The cognitive load shift is different — the user doesn't manage multiple agents; the single agent manages the user's multiple browser contexts. The dispatcher metaphor breaks: it's closer to hiring one executive assistant who reads all your open documents rather than multiple specialists each holding one.
