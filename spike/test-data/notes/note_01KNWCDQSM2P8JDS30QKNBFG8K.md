ChatGPT 搜索的动态呈现定制化网页目标，是 Generative UI 范式从 AI 协作工具向搜索结果呈现的延伸。现有知识库把 Generative UI 定义为 AI 同时设计界面和内容，讨论的是 AI 助手输出侧的 UI 生成。而 ChatGPT 搜索的方向是：搜索结果本身不再是蓝链列表，而是一个为当前查询动态生成的定制页面。这意味着 Generative UI 的应用范围从人机交互界面扩展到信息检索呈现——搜索结果不再是检索到的文档，而是生成的界面。这进一步模糊了搜索引擎与 AI 助手的边界：当搜索结果本身是生成的，搜索就成为了生成的一个触发器。
