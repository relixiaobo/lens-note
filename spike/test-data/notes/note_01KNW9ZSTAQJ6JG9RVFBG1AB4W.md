The 'Proactive' dimension of Google Assistant's design is structurally distinct from — and actually deeper than — Ambient UX. Ambient UX (see note_01KNW55PVTPAR5BZEGN4R3J2V0) is about reducing the attention cost of *execution*: the user still formulates a request, the agent just executes it silently. Proactive AI goes one layer further: it eliminates the need for the user to formulate a request at all. The agent reads the calendar, infers an upcoming need (a physics exam), and prepares materials without any prompt. This is not just Ambient UX — it's the agent taking over the *intent-formulation* step, not just the execution step. The user-agent relationship shifts from 'user directs, agent executes' to 'agent anticipates, user approves or ignores.'
