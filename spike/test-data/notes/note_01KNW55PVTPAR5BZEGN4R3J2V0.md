Ambient UX (AI working silently in the background, interrupting only with results) represents a different theory of human attention than Collaborative or Spreadsheet UX. Spreadsheet UX assumes the user wants to orchestrate and observe. Collaborative UX assumes the user wants to co-create. Ambient UX assumes the user's attention is the scarce resource — it treats focus as something to protect, not a channel to exploit. This maps cleanly onto the distinction between 'push' and 'pull' information architectures: ambient agents push only summaries and diffs, never demanding real-time engagement.
