Gemini in Chrome automating web actions (booking appointments, ordering groceries) on behalf of the user represents the Ambient UX model pushed to its logical extreme: not just the AI's *output* being ambient, but the AI's *execution* being ambient. The user states an intent and returns to other things — the web interaction itself happens without them. This is a sharper instantiation of the principle in note_01KNW55PVTPAR5BZEGN4R3J2V0 that 'user attention is the scarce resource.' The browser agent doesn't just protect attention from AI output overhead — it eliminates the attention cost of entire classes of routine digital labor.
