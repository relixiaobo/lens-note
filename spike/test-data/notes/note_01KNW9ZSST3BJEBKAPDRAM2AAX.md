Google's 3P framework (Personal, Proactive, Powerful) bundles three dimensions that are actually in tension with each other. Being highly Personal requires deep data access, which reduces user privacy autonomy. Being highly Proactive means the agent decides when and how to act, which reduces user agency over initiation. Being highly Powerful expands the blast radius of any error. Google's framing presents these as additive virtues, but a more honest rendering reveals they trade against each other: the more Proactive and Personal an agent becomes, the more dangerous being Powerful is without very robust user control mechanisms.
