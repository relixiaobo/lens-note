Emerging empirical work begins to close the measurement gap: Tornhill & Borg (2024) found that low-quality code takes more than twice as long to resolve issues and has 15× higher defect density. If replicated at scale, this converts the internal-quality argument from developer intuition into hard organizational evidence — which changes the rhetorical terrain significantly.
