Chat UX is notably absent from the four-paradigm Agent UX taxonomy (Spreadsheet, Collaborative, Ambient, Generative). This article suggests Chat deserves its own axis: it is the only paradigm where the *communication channel itself* — not the task output — is the primary artifact. The others deal with what the agent produces; Chat deals with the conversational negotiation of what the agent should produce. This makes Chat a meta-layer, not just another interaction mode.
